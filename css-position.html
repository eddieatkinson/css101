Position property has 4 options

  1. Static
  2. Relative
  3. Fixed
  4. Absolute

Static

Static is the default position for the position property.

Top, bottom, left, and right do NOT work with static.

This element will show up, top and bottom, left and right. Nothing special, and it can;t be moved.

Static box

Relative

A relative position is the same as static (top to bottom, left to right), EXCEPT you can move it with top, bottom, left, and right.

IF you move it, it will sit on top of other stuff and the rest of the page will think it's still where "static" would be

Relative box

Fixed

A fixed position element will remain relative to the viewport (screen). Even if the user scrolls, the screen doesn't move, so the element doesn't move either.

Use top, bottom, left, right to place the element

Unlike relative elements, it does not leave a gap, i.e., the rest of the page (static, relative things) don't know it exists anymore.

Fixed box

absolute

Absolute position elements work off of the grid system. Just like they are ignored by everyone. They can have a z-index. They do not leave space/gaps.

The top, bottom, left, and right are determined by the first tag up that has a position set.
